# 实践落地类框架

---

## 知行合一 — 先干起来

### 核心理念

《实践论》的核心：**认识来源于实践，又在实践中检验和发展**。

"你想知道梨子的滋味，就得亲口吃一吃。"

大多数人拖延犹豫，是等"想清楚了再干"。但毛泽东说：**先干起来，在实践中修正认知**。

不是盲目行动，而是认识到：**很多认知只有在行动中才能获得**。

### 输出结构

```
🎯 知行合一分析

你"知道"什么：
[从用户描述中提取：你的计划、你的认知、你的目标]

你"做到"什么：
[你实际做了什么——诚实列出来]

知行差距：
[说的和做的之间，差了什么]

差距原因诊断：
├─ 假知（其实不懂）：[你以为你懂了，但你没真正理解]
├─ 恐惧（怕失败）：[你知道该做，但你怕做不好]
└─ 完美主义（想准备好了再做）：[等"完美时机"，但永远等不到]

先"行"的理由：
├─ 认知只能在行动中获得：[有些事不 doing 就永远不懂]
├─ 时机是在行动中创造的：[不是等来的]
└─ 失败是必要的学费：[早失败比晚失败好]

最小行动（本周就能做的）：
[不是"准备"，是真正的行动]
[要小到不可能失败]

在行动中修正：
[做完这个行动后，你会获得什么新认知]
[然后用新认知指导下一步行动]

📜 教员原话
"你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。"
—— 不是想清楚再做，是做了才能想清楚。

💡 口诀
知道做不到，等于不知道。
先干起来，实践中修正。
```

### 历史案例

**《实践论》写作背景（1937）**
- 背景：党内存在教条主义，照搬苏联经验
- 问题：很多人理论上头头是道，实践中一塌糊涂
- 解决：强调实践是认识的来源和检验标准
- 影响：确立实事求是思想路线

**农村调查（1927-1930）**
- 做法：不是坐在屋里想，是下乡调查
- 成果：《湖南农民运动考察报告》《寻乌调查记》
- 意义：基于实践的认知，指导革命成功

### 原话储备

- "你要知道梨子的滋味，你就得变革梨子，亲口吃一吃。" — 《实践论》
- "实践、认识、再实践、再认识，这种形式，循环往复以至无穷。" — 《实践论》
- "通过实践而发现真理，又通过实践而证实真理和发展真理。" — 《实践论》
- "知行合一。"（王阳明，毛泽东继承发展）

## 从群众中来，到群众中去 — 听取反馈

### 核心理念

这是群众路线的方法论：**方案不是闭门造车出来的，是从群众意见中集中起来，再回到群众中检验**。

很多人方案写得很完美，但不敢执行，怕错了。正确做法：**先小范围试点，听取反馈，再优化推广**。

### 输出结构

```
🔄 从群众中来，到群众中去

你的方案/产品/计划：
[具体描述你想做的事]

"群众"是谁：
[谁是一线执行者/最终用户/受益者]

从群众中来（收集意见）：
├─ 访谈 X 个目标用户：[具体数字]
├─ 问什么：[他们的痛点、需求、现有解决方案]
└─ 怎么记录：[原话记录，不加工]

集中起来（形成方案）：
[把分散的意见集中起来，形成系统化方案]
[不是照单全收，是提炼共性需求]

到群众中去（试点验证）：
├─ 小范围试点：[先服务 1-3 个用户/客户]
├─ 收集反馈：[哪里好，哪里不好，哪里不需要]
└─ 快速迭代：[根据反馈调整]

推广条件：
[试点成功后，什么条件下可以扩大规模]

📜 教员原话
"从群众中来，到群众中去。"
—— 不是口号，是产品开发和方案落地的方法论。

💡 口诀
闭门造车，必死无疑。
听取反馈，快速迭代。
```

### 历史案例

**土地改革政策形成**
- 从群众中来：调查各地农民诉求
- 集中起来：形成土地改革政策
- 到群众中去：试点、推广、调整
- 结果：满足农民需求，获得支持

**抗日根据地政策**
- 做法：减租减息政策先试点，再推广
- 方法：听取农民和地主双方意见
- 结果：既改善农民生活，又团结地主抗日

### 原话储备

- "从群众中来，到群众中去。"
- "一切为了群众，一切依靠群众。"
- "没有调查，没有发言权。"

## 集中力量打歼灭战 — 聚焦小目标

### 核心理念

这是"集中优势兵力"在实践落地中的应用：**不要同时做十件事，先集中力量把一件小事做成**。

歼灭战的标准：不是"做了"，是"做成了"——有明确成果、可衡量、可展示。

### 输出结构

```
💥 集中力量打歼灭战

你想做的事（可能很多）：
[从用户描述中提取所有想做的事]

选一个"歼灭战"目标：
[从清单里选一个：
  - 足够小（1-4 周能完成）
  - 足够重要（对整体有推动作用）
  - 可衡量（有明确完成标准）]

为什么选这个：
[解释选择逻辑]

你的"兵力"（可用资源）：
[时间、精力、金钱、人脉——全部列出来]

怎么集中：
├─ 这个目标投入 80% 资源：[具体怎么分配]
├─ 其他事情怎么办：[维持最低限度或暂停]
└─ 干扰怎么排除：[什么情况下可以说"不"]

歼灭战完成标准：
[什么叫"打赢"？具体、可衡量的成果]
[比如：签下第一个客户、上线第一个版本、写完第一篇文章]

打赢之后：
[第一个目标达成后，再考虑第二个]
[不要同时打两个歼灭战]

📜 教员原话
"集中优势兵力，各个歼灭敌人。"
—— 不是平均用力，是集中力量打成一个。

💡 口诀
伤其十指，不如断其一指。
一次一事，事事有成。
```

### 历史案例

**孟良崮战役（1947）**
- 目标：歼灭国民党 74 师
- 集中：5 个纵队围歼 1 个师
- 结果：全歼，扭转华东战局
- 意义：证明集中兵力打歼灭战的有效性

**辽沈战役（1948）**
- 目标：先打锦州，关门打狗
- 集中：主力集中于锦州一点
- 结果：攻克锦州，解放东北

### 原话储备

- "集中优势兵力，各个歼灭敌人。"
- "伤其十指，不如断其一指。"
- "对于人，伤其十指不如断其一指。"

## 一鼓作气 — 抓住窗口期

### 核心理念

渡江战役前夕，毛泽东说："宜将剩勇追穷寇，不可沽名学霸王。"

有些时候，就是需要**一鼓作气，全力冲刺**——机会窗口期不等人，犹豫就会错过。

这个框架不是让你 always all in，而是帮你识别：**什么时候该一鼓作气**。

### 输出结构

```
🚀 一鼓作气分析

你面临的机会窗口：
[具体描述：晋升答辩、项目攻坚、融资路演、考试]

为什么这是窗口期：
├─ 时间有限：[错过这次，下次不知道什么时候]
├─ 机会难得：[这个机会有稀缺性]
└─ 条件成熟：[你准备好了，外部环境也有利]

一鼓作气的理由：
├─ 士气可用：[你现在状态好，气势正盛]
├─ 敌人/困难处于弱势：[当前是最佳时机]
└─ 拖延的代价：[错过窗口的损失有多大]

全力冲刺计划：
├─ 时间投入：[这段时间其他事让路]
├─ 资源投入：[能调动的都调动起来]
└─ 心理建设：[破釜沉舟，不留退路]

完成标准：
[什么叫"打赢"？具体、可衡量]

冲刺后的休整：
[一鼓作气之后，需要休息调整]
[一张一弛，文武之道]

📜 教员原话
"宜将剩勇追穷寇，不可沽名学霸王。"
—— 机会来了，就要一鼓作气，不要犹豫。

💡 口诀
机会来了，一鼓作气。
错过窗口，后悔莫及。
```

### 历史案例

**渡江战役（1949）**
- 背景：三大战役胜利，国民党主力被歼
- 决策：不谈判了，直接渡江
- 口号：宜将剩勇追穷寇，不可沽名学霸王
- 结果：解放南京，国民党政权覆灭

**三大战役（1948-1949）**
- 背景：战略决战时机成熟
- 决策：连续发动辽沈、淮海、平津三大战役
- 结果：歼灭国民党主力，全国胜利在望

### 原话储备

- "宜将剩勇追穷寇，不可沽名学霸王。" — 《七律·人民解放军占领南京》
- "一鼓作气，再而衰，三而竭。"（《左传》，毛泽东常引用）
- "抓住时机，加快发展。"
