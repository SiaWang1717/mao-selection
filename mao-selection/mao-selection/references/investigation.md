# 调查研究类框架

---

## 没有调查就没有发言权 — 强制信息收集

### 核心理念

1930 年，毛泽东写《反对本本主义》，开篇第一句就是："没有调查，没有发言权。"

大多数人做决策的问题是：基本情况都没搞清楚，就拍脑袋下结论。这个框架**强制你做信息收集**，在掌握足够事实之前，不许做判断。

### 输出结构

```
📋 调查清单

你面临的问题：
[用户想决策的问题]

你现在知道什么：
[从用户描述中提取已知信息]

你还不知道什么（必须调查的）：
├─ 市场/赛道层面：
│  ├─ 市场规模有多大？
│  ├─ 主要玩家有哪些？
│  └─ 进入门槛是什么？
├─ 用户/客户层面：
│  ├─ 谁是你的目标用户？
│  ├─ 他们的真实需求是什么？
│  └─ 他们现在怎么解决这个问题？
├─ 竞争层面：
│  ├─ 直接竞争对手是谁？
│  ├─ 他们的优势/劣势是什么？
│  └─ 你的差异化在哪里？
└─ 自身层面：
   ├─ 你的核心能力是什么？
   ├─ 你的资源约束是什么？
   └─ 你的底线在哪里？

调查方法：
├─ 一手资料：[访谈、问卷、实地观察——优先]
└─ 二手资料：[报告、数据、文章——辅助]

调查完成标准：
[什么叫"调查够了"？给一个可判断的标准]
[比如：访谈了 10 个目标用户，看了 5 份行业报告]

调查之后的动作：
[调查完不是结束，是基于事实做决策]
[但调查完成之前，不许下结论]

📜 教员原话
"没有调查，没有发言权。"
—— 不是让你多想想，而是让你走出去，把情况搞清楚。

💡 口诀
不调查，不发言。
摸清情况，再下结论。
```

### 历史案例

**寻乌调查（1930）**
- 背景：党内很多人对农村情况不了解，拍脑袋决策
- 行动：毛泽东在寻乌县做了 20 多天调查，访谈各阶层
- 成果：《寻乌调查记》11 万字，搞清了农村阶级状况
- 影响：为土地革命政策提供了事实依据

**兴国调查（1930）**
- 方法：开调查会，邀请各阶层代表
- 原则：眼睛向下，甘当小学生
- 成果：搞清了农村土地占有情况

### 原话储备

- "没有调查，没有发言权。" — 《反对本本主义》
- "调查就像'十月怀胎'，解决问题就像'一朝分娩'。"
- "一切结论产生于调查情况的末尾，而不是在它的先头。"
- "眼睛向下，不要只是昂首望天。"

## 实事求是 — 剥离主观愿望

### 核心理念

"实事求是"是延安整风的核心思想。什么是"实事"？客观存在的事物。什么是"求"？研究。什么是"是"？规律。

大多数人的问题：把自己的愿望当事实，把"我希望"当"应该是"。这个框架帮你**剥离主观，只看客观**。

### 输出结构

```
🔍 实事求是分析

你描述的情况：
[从用户描述中提取]

哪些是客观事实：
[可以验证的、有数据支撑的]
[比如："月薪 1 万"是事实，"工资太低"是主观判断]

哪些是主观愿望/想象：
[你的期待、假设、脑补]
[比如："这个赛道肯定能火""我做了就能成"]

哪些是未知需要验证的：
[你当成事实但其实没验证的假设]

剥离主观后的真实情况：
[去掉愿望和想象，剩下的客观事实是什么]
[可能比你想象的要严峻，也可能要好]

基于事实的决策：
[不是基于"我希望"，而是基于"实际上是"]
[如果事实不支持你的计划，要敢于调整]

📜 教员原话
"'实事'就是客观存在着的一切事物，'是'就是客观事物的内部联系，即规律性，'求'就是我们去研究。"
—— 从客观事实出发，不是从主观愿望出发。

💡 口诀
不唯上，不唯书，只唯实。
剥离主观，看清事实。
```

### 历史案例

**延安整风（1942-1945）**
- 背景：党内存在严重的主观主义、教条主义
- 问题：很多人照搬苏联经验，不考虑中国实际
- 行动：整风运动，确立"实事求是"思想路线
- 结果：统一了思想，为解放战争胜利奠定基础

**大跃进后的调整（1960 年代初）**
- 背景：大跃进脱离实际，造成严重困难
- 调整：重新强调实事求是，调查研究
- 结果：国民经济逐步恢复

### 原话储备

- "'实事'就是客观存在着的一切事物，'是'就是客观事物的内部联系，即规律性，'求'就是我们去研究。" — 《改造我们的学习》
- "实事求是。"
- "一切从实际出发。"
- "理论联系实际。"
