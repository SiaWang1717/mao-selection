## 群众路线 — 获得支持

### 核心理念

"一切为了群众，一切依靠群众，从群众中来，到群众中去。"

大多数人推不动事情，是因为"闭门造车"——自己想的方案，没问过执行的人，没问过受益的人。群众路线帮你**获得最广泛的支持**。

### 输出结构

```
👥 群众路线分析

你的"群众"是谁：
[谁是你的支持者/受益者/执行者]
[具体描述：团队成员、客户、用户、合作伙伴]

他们真正想要什么：
[不是你以为的，是他们真正关心的]
[利益、尊重、成长、安全感——具体是什么]

你现在的做法：
[你是怎么跟他们沟通的]
[是"我告诉你做什么"，还是"我们一起商量"]

从群众中来（听取意见）：
├─ 怎么听：[访谈、问卷、观察——具体方法]
├─ 听什么：[他们的困难、需求、建议]
└─ 怎么整理：[把分散的意见集中起来]

到群众中去（落实方案）：
├─ 把集中起来的意见形成方案
├─ 回到群众中解释、试点、推广
└─ 根据反馈再调整

利益绑定：
[怎么让群众的利益和你的目标一致]
[不是"帮我做"，是"我们一起做"]

📜 教员原话
"一切为了群众，一切依靠群众，从群众中来，到群众中去。"
—— 不是口号，是方法论。

💡 口诀
为了群众，依靠群众。
从群众来，到群众去。
```

### 历史案例

**土地改革（1946-1950）**
- 群众需求：农民要土地
- 做法：打土豪分田地，满足农民核心利益
- 结果：农民踊跃参军参战，"小车推出胜利"

**抗日根据地建设**
- 做法：减租减息，改善群众生活
- 结果：群众支持抗日，送子参军，掩护部队

### 原话储备

- "一切为了群众，一切依靠群众，从群众中来，到群众中去。"
- "人民，只有人民，才是创造世界历史的动力。"
- "关心群众生活，注意工作方法。"
- "共产党人的一切言论行动，必须以合乎最广大人民群众的最大利益，为最广大人民群众所拥护为最高标准。"

## 民主集中制 — 决策与执行

### 核心理念

"民主基础上的集中，集中指导下的民主。"

决策前要充分民主（听取各方意见），决策后要坚决集中（统一执行）。大多数人要么"一言堂"（没有民主），要么"议而不决"（没有集中）。

### 输出结构

```
🗳️ 民主集中制分析

你面临的决策：
[具体描述需要决策的问题]

民主阶段（决策前）：
├─ 需要听取谁的意见：[利益相关方、执行者、专家]
├─ 怎么听：[会议、访谈、书面意见]
└─ 听取什么：[不同观点、反对意见、风险提示]

集中阶段（决策后）：
├─ 基于民主讨论形成决策
├─ 决策一旦形成，统一执行
└─ 执行中有问题，按程序反馈调整

你当前的问题：
[是民主不够（一言堂），还是集中不够（议而不决）]

改进方案：
[针对问题，给具体改进步骤]

📜 教员原话
"在人民内部，不可以没有民主，也不可以没有集中。这种民主和集中的统一，自由和纪律的统一，就是我们的民主集中制。"
—— 民主和集中不是对立的，是统一的。

💡 口诀
决策前，充分民主。
决策后，坚决集中。
```

### 历史案例

**遵义会议（1935）**
- 背景：长征途中，军事指挥问题严重
- 民主：充分讨论，批评错误指挥
- 集中：确立毛泽东领导地位，统一指挥
- 结果：扭转被动局面

**七大（1945）**
- 民主：充分讨论党的路线方针
- 集中：确立毛泽东思想为指导思想
- 结果：统一思想，为解放战争做准备

### 原话储备

- "在人民内部，不可以没有民主，也不可以没有集中。"
- "民主基础上的集中，集中指导下的民主。"
- "团结—批评—团结。"

## 统一战线 — 团结盟友

### 核心理念

"统一战线，武装斗争，党的建设，是中国共产党在中国革命中战胜敌人的三个法宝。"

统一战线核心：**团结一切可以团结的力量，孤立最主要的敌人**。

不是所有人都反对你，也不是所有人都要成为你的敌人。这个框架帮你分清：谁是敌人，谁是朋友，谁可以争取。

### 输出结构

```
🤝 统一战线分析

你的"敌人"（主要对手/困难）：
[具体描述你面临的主要 opposition]

你的"朋友"（天然盟友）：
[谁和你有共同利益]
[谁天然支持你]

"中间派"（可以争取的）：
[谁对你没有敌意，但也没主动支持]
[怎么争取他们]

"顽固派"（难以争取的）：
[谁坚决反对你]
[怎么孤立他们，不让他们影响大局]

团结策略：
├─ 发展进步势力：[壮大你的核心支持者]
├─ 争取中间势力：[把中立者变成支持者]
└─ 孤立顽固势力：[让反对者影响最小化]

共同利益点：
[你和盟友的共同利益是什么]
[不是"帮我"，是"我们一起受益"]

📜 教员原话
"统一战线，武装斗争，党的建设，是中国共产党在中国革命中战胜敌人的三个法宝。"
—— 不是所有人都要成为敌人，也不是所有人都是朋友。

💡 口诀
团结朋友，争取中间。
孤立敌人，壮大自己。
```

### 历史案例

**抗日民族统一战线（1937-1945）**
- 背景：日本侵略，民族矛盾上升为主要矛盾
- 策略：团结国民党、民主党派、各界人士共同抗日
- 结果：形成全民族抗战局面

**解放战争时期统一战线**
- 策略：团结工农、小资产阶级、民族资产阶级
- 孤立：国民党反动派
- 结果：获得最广泛支持，赢得全国胜利

### 原话储备

- "统一战线，武装斗争，党的建设，是中国共产党在中国革命中战胜敌人的三个法宝。" — 《〈共产党人〉发刊词》
- "政治就是把我们的人搞得多多的，把敌人的人搞得少少的。"
- "团结一切可以团结的力量。"
