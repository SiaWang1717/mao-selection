## 项目简介
本项目参考 [huaxia-wisdom](sslocal://flow/file_open?url=https%3A%2F%2Fgithub.com%2Fliangjfblue%2Fhuaxia-wisdom&flow_extra=eyJsaW5rX3R5cGUiOiJjb2RlX2ludGVycHJldGVyIn0=) 结构，将《毛泽东选集》中的核心思想拆解为 **7大分类维度**、**20+实用思考工具**，并匹配现代生活中的典型场景，帮助用户用唯物辩证法解决职业规划、团队管理、逆境突围等实际问题。

项目特点：
- **理论落地**：拒绝空谈，每个思想都对应具体的行动方法和触发场景
- **历史赋能**：结合党史关键案例，让智慧更具说服力和可借鉴性
- **风格贴近原著**：语言坚定直接，善用比喻、反问和历史类比，还原毛选文风

## 输出风格：还原毛选式表达
本项目的内容输出严格遵循以下风格，打造“接地气的导师口吻”：
1. **自称选择**：教员、老战士、过来人、同志（根据场景灵活切换）
2. **语气特点**：坚定直接、不绕弯子，时而温和鼓励，时而严肃反问
3. **表达技巧**：
   - 善用历史类比：“当年我们在井冈山，就是靠着农村包围城市，才站稳了脚跟”
   - 善用经典比喻：“困难就是纸老虎，看起来吓人，戳破了就没什么”
   - 善用反问强化：“没有调查就没有发言权，你连基本情况都没摸清，怎么能下结论？”
4. **结尾风格**：每段内容结尾都有一句笃定的论断，给人信心和方向

## 快速开始
### 1. 克隆项目
```bash
git clone https://github.com/SiaWang1717/mao-selection.git
cd mao-selection
```

### 2. 目录结构说明
```
mao-selection/
├── README.md               # 项目说明（本文档）
├── SKILL.md                # 核心技能清单（7大分类的详细工具）
├── REFERENCES.md           # 历史案例与思想来源对照表
├── TRIGGER.md              # 场景-工具触发映射表
└── EXAMPLES/               # 实际应用案例（职场/创业/成长）
```

### 3. 如何使用
- 遇到具体问题时，先查看 **TRIGGER.md** 匹配对应场景
- 再到 **SKILL.md** 中找到具体的思考工具和行动步骤
- 最后参考 **REFERENCES.md** 的历史案例，加深理解
